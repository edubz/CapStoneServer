The embedded video is a live stream of an audio system made in Max 8, which involves using web servers to collect user submitted audio files from this website and a limited amount of custom raspberry pi devices.

The audio files are then processed through variety of effects with parameters that can be controlled wirelessly via a limited amount of collectible sensor-filled devices.

This website, the Max patch, and all microcontroller devices designed, handcrafted, and maintained by PCC Creative Coding Capstone students. Source code here
